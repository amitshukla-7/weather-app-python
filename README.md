# weather-app-python
In this project, i have created a small weather app that tells the weather of any city in the world

A simple desktop weather application built using Python and PyQt5. The app fetches real-time weather data from the OpenWeatherMap API and displays the temperature, weather description, and a matching emoji for better visual feedback.

#Features
- Simple and clean PyQt5 user interface  
- Real-time weather data  
- Temperature shown in Celsius  
- Weather description and emoji based on conditions  
- Handles invalid city names and network errors  

#Technologies Used
- Python  
- PyQt5  
- Requests  
- OpenWeatherMap API
